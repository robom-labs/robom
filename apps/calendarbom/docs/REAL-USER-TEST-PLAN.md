# 실제 사용자 테스트 계획 (사람 작업 — 미실시, 검증 예정)

자동화 테스트만으로 "70대가 쉽게 쓴다"고 선언하지 않는다. 아래 테스트는 아직 실시하지 못했다.

## 표본

- 65세 이상 3명 이상(가능하면 70대 포함), 20~40대 3명 이상.

## 과제 (안내 없이)

1. (70대) 다음 주에 병원 가는 날을 달력에 넣고 알람을 받아보세요.
2. (70대) 매일 아침·저녁 약 알람을 만들어 보세요. 아침 약을 "먹었어요"로 기록해 보세요.
3. (70대) 손주 생일을 등록해 보세요.
4. (전연령) 방금 만든 병원 일정을 다음 달로 하나 더 만들어 보세요. (지난번과 같게/다음 일정)
5. (전연령) 병원 일정을 가족에게 문자로 보내보세요.

## 관찰 기록 양식

| 항목 | 기록 |
|---|---|
| 처음 누른 곳 | |
| 멈춘 곳(3초 이상) | |
| 질문을 다시 읽은 횟수 | |
| 잘못 누른 곳 | |
| 도움 요청 여부 | |
| 완료 시간 | |
| 사용자가 직접 한 말 | |

## 실패 시 우선 수정 순서

1. 멈춘 화면의 질문 문구 (쉬운 한국어로 재작성)
2. 첫 화면 칩 순서·이름
3. 시간 선택 빠른 칩 구성
4. 저장 후 확인 문장

## 데이터 취급

- 테스트에는 가상의 샘플 이름(엄마·민수·지연)만 사용한다. 실제 약 이름·개인 일정을 기록에 남기지 않는다.
