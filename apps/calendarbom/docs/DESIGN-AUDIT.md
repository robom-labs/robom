## 1차 감사 — v0.1.0 화면

| 점검 항목 | 판정 | 근거·조치 |
|---|---|---|
| 무엇을 먼저 봐야 하는지 명확한가 | 통과 | 월 제목→달력→다가오는 일정 위계 유지 |
| 날짜 숫자가 충분히 큰가 | 통과 | 1.1rem(글자 크게 시 확대) |
| 오늘·선택 날짜 구분 | 통과 | 오늘=연보라 배경+테두리, 선택=진보라 테두리 |
| 월 이동 버튼 크기 | 통과 | 56px(320px에서 48px) |
| 버튼이 지나치게 많이 보이는가 | **실패** | 시트에 시 12+분 6+알람 5+내용 9칩 동시 노출 → 아코디언+시간 재설계로 수정 |
| 저장 버튼 위치 고정 | **실패** | 긴 시트 하단으로 밀림 → sticky 저장으로 수정 |
| 설명 문장이 긴가 | 통과 | 질문형 한 문장 유지 |
| 여러 질문 동시 노출 | **실패** | 1·2·3 전부 펼쳐짐 → 현재 질문만 펼침 |
| 약·반복 때문에 달력 과밀 | 해당 없음(v0.1.0) → V6에서 약은 묶음 1표시 규칙 적용 |
| 하단바가 콘텐츠 가림 | 통과 | padding-bottom 108px+safe-area |
| 큰 글자에서 겹침 | 통과(스윕 확인) | xl+320px 재검증 예정 |
| 젊은 사용자에게 촌스러운가 | 주의 | 장식 없는 라벤더 톤은 중립적. 시트 과밀이 "옛날 폼" 인상 → 아코디언이 해결 |
| 패밀리 일관성 | 통과 | 워드마크·하단바·설정 킷 정본 준수 |
