# 경쟁 제품 단순화 감사 (V6)

조사 방법 정직 보고: 이 실행 환경은 외부 페이지 직접 열람이 차단되어(프록시 정책),
WebSearch 결과 요약(Google Calendar 반복·기본 알림, Apple Health Medications 흐름, WCAG 2.2 목표 크기)과
학습 지식을 근거로 작성했다. 실행 시점에 확인한 출처:
support.google.com/calendar/answer/37115 · support.apple.com/en-us/105064 ·
w3.org/WAI/WCAG22/Understanding/target-size-minimum. 화면·문구·아이콘·색은 복사하지 않았다.

## WCAG 2.2 근거 반영

- 목표 크기: AA 24px, AAA 44px — 캘린더봄은 주요 컨트롤 48px+ 유지(AAA 초과).
- 인지 접근성(WCAG 2.2 redundant entry): **이미 입력한 정보를 다시 묻지 않음** → 최근 사용·지난번과 같게·기본값 제안의 규범 근거.
