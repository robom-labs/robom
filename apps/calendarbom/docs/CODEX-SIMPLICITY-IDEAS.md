## 자동 구현 2개의 증명 (질문·화면 복잡도 무증가)

1. **임시 입력 자동 복구** — 시트를 닫으면 `calendarbom:draft:v2`에 초안 저장, 같은 날짜 재진입 시 이어서 표시.
   질문 0개 추가(자동), 화면 요소 0개 추가(토스트 안내 1회). 증거: E2E "초안 복원: 민수와 약속 이어서".
2. **저장·삭제 후 되돌리기** — 모든 파괴적 동작 전 스냅샷, 토스트에 [되돌리기] 8초.
   질문 0개 추가(확인 팝업을 오히려 제거 — v0.1.0의 confirm() 삭제). 증거: E2E "삭제 되돌리기: 일정 복구".

#3·#4는 v6 계약 본문(§4-1·4-2, TIME/FLOW ID)이 직접 요구한 항목이라 "추가 아이디어 2개 한도"와 별도로 구현했다.
