# 로봄 패밀리 작업 컨텍스트

- `robom`은 registry·패밀리 규격·설치 허브·관제의 정본입니다.
- 앱 기능과 데이터, 빌드·배포·롤백은 각 앱 저장소가 소유합니다.
- 앱은 중앙 저장소를 런타임이나 빌드 시 내려받지 않고 커밋된 생성물을 사용합니다.
- 공통화 대상은 워드마크, 토큰, 앱바·하단 메뉴 geometry, 설정 큰 순서, 상태·설치·분석·광고 계약입니다.
- 앱별 핵심 기능, 색, 정보 순서, 추천·판단 로직은 공통화하지 않습니다.
- 정확한 위치, 일정 내용, 약·병원, 청약 조건, 검색어 원문, OAuth·푸시 비밀은 분석 이벤트로 보내지 않습니다.
- 배포 전 생성물 drift, 저장 데이터 migration, 모바일 320px·200% 확대, PWA·서비스워커, 운영 URL 빌드 SHA를 확인합니다.
