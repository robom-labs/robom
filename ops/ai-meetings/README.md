실행형 요청은 조사부터 구현, 검증, PR, 병합, 기존 배포 파이프라인과 운영 검증까지 연속 수행한다. 유료 지출, 계정·청구·권한, 비밀값, 파괴적 데이터 작업, 법적 계약과 신원 확인은 예외다. 세부 규칙은 `PROTOCOL.md`와 `COMPANY-MODE.md`를 따른다.
